# Constructor - modified

Constructor project is a toolkit for building newsletter campaigns and landing pages. It ships with a campaign creator tool, a revised UX/UI, and a custom translation API to support localized content.

## Quick start

1. Set the vite scope to load a specific campaign folder:
    - Example: `VITE_SCOPE=KamilK`
    - This loads campaigns from 0-level in `./campaigns/KamilK/`, eg. skipping `./campaigns/KamilK/archive/`.
2. `pnpm dev`

## Key features

- Create and preview newsletter campaigns and landing pages
- Custom translation API for multi-language content
- Vite-scope campaign loading for scoped development
- Integrated campaign creator tool and refreshed UX/UI
